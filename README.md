## Лабораторна робота №7
# Message Queue Chat Client

## Запуск сервера

1. Перейти в папку `server.mq`
2. Виконати:

```
mvn clean compile exec:java -Dexec.args="0.0.0.0 61616"
```

Або:

```
start.cmd
```

## Запуск клієнта

Запустити `Main.java` в IntelliJ IDEA.

Для тестування можна запускати кілька клієнтів одночасно.

## Доступні команди

```
ping
echo <text>
login <username> <password>
list
msg <username> <message>
file <username> <path>
drain
exit
```

## Опис роботи

* `ping` — перевірка з’єднання з сервером
* `echo` — відправка тексту і отримання його назад
* `login` — реєстрація або вхід користувача
* `list` — список користувачів
* `msg` — відправка повідомлення іншому користувачу
* `file` — відправка файлу
* `drain` — примусове отримання та видалення повідомлень/файлів
* `exit` — вихід з програми

## Особливості

- Використано **Java Message Service (JMS)** для роботи з повідомленнями    
- Використано **ActiveMQ** як брокер повідомлень    
- Реалізовано **асинхронне отримання повідомлень** через `MessageListener`    
- Передача файлів реалізована через `ObjectMessage`    
- Використано окремі сесії для синхронного та асинхронного обміну
